Desenvolvedoras do site:

Este primeiro projeto do programa TWIST 2024.2 tem como objetivo trabalhar com formulários envolvendo o uso de expressões regulares para buscar requisições específicas em um texto. Por exemplo, iremos buscar certos padrões e sua quantidade de ocorrências (como quantidade de caracteres especiais) e restringir tipos de arquivo para envio.

As ferramentas utilizadas neste projeto foram: a linguagam de marcação HTML, a linguagem para estilização CSS e a linguagem de programação JavaScript.
